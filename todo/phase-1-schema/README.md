# Phase 1: Wiki Schema Extension

**Status:** `active`
**Prerequisites:** Phase 0
**Spec reference:** `docs/wiki-schema.md`

## Purpose

Migriere das alte flache Schema auf das neue Dual-State-Schema (K0/K1) mit klaren Typisierungen für Entitäten, Konzepte, narrative States und den Reader State.
