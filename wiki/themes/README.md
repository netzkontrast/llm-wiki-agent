# Themes

Thematic thread pages. Each page tracks one theme: its statement, expression across plot/character/setting, motifs, evolution across the novel, and dialectical relationship to counter-themes.

## Pages
- [IntegrationVsFragmentation.md](IntegrationVsFragmentation.md) — central dialectical theme, drives all conflicts and arcs

## Connections
- Related to: `wiki/conflicts/` — conflicts are thematic engines; they link to the themes they express
- Related to: `wiki/characters/` — character arcs embody thematic transformations
- Related to: `wiki/foreshadowing/` — motifs listed in theme pages are tracked as foreshadowing strands
- Related to: `wiki/chapters/` — chapters list active themes in `themes:` field

## Routing Layer
Part of the **Narrative Layer** — see [`wiki/narrative/README.md`](../narrative/README.md) for loading rules and mutation constraints.

## Conventions
- Naming: `TitleCase.md` (e.g., `IntegrationVsFragmentation.md`, `ParadoxAsFeature.md`)
- Temporal: No — themes are timeless; they evolve across the story but the page tracks that evolution via `evolution_stages:`.
