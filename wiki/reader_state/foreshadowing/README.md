## Conventions
- Naming: `kebab-case.md` (e.g., `the-cage-motif.md`, `aegis-true-nature.md`)
- Temporal: No — pages track the full lifespan of a strand; status field records current state
- Unresolved strands: If a strand remains unresolved at story completion, add a note flagging it as `intentionally-unresolved` and document the narrative reason
