# Wiki Root

This is the entry point for the wiki structure. The wiki is organized into 4 main routing layers to optimize agent context loading:

## Layers
- `knowledge/` — Source-derived facts, real-world entities, concepts, rules, timeline
- `narrative/` — Story structure: characters, chapters, locations, conflicts, themes, arcs, dramatica, beats, outlines, manuscripts
- `reader_state/` — Progressive disclosure: reader-model, foreshadowing
- `meta/` — System admin: archive, ingest, logs, contradiction-log, protocols

## Key Files
- `index.md` — Catalog of all pages
- `log.md` — Chronological operation log
- `overview.md` — Living synthesis
